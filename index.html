Welcome to uxVirtual!

This site is a multi-room virtual environment compatible with Oculus Rift and JanusVR. You'll need to download JanusVR 28 or higher to view it.

Currently the site has the following rooms:

  • Home room - The central hub to which other rooms link to.
  • Destiny promotional room - A sample promotional room for the Destiny game for next gen and current gen game consoles.
